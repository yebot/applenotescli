# Apple Notes CLI

A command-line tool for CRUD operations on Apple Notes.

## Installation

```bash
uv pip install -e .
```

## Usage

```bash
notes --help
notes list
notes show "Note Title"
notes create "New Note"
notes delete "Note Title"
```

## Requirements

- macOS
- Python 3.10+
- Terminal must be granted access to Notes.app (System Preferences > Privacy & Security > Automation)
